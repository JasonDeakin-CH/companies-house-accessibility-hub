Accessibility checklist

  1. Write in plain language, using simple sentences.
  2. Write descriptive links and headings.
  3. Organise content linearly, logically and consistently.
  4. Break up content with sub-headings and lists (bulleted/numbered).
  5. Make important information stand out.
  6. Align content to the left and avoid all capitals.
  7. Use good colour contrast and a readable font size and font type.
  8. Use images (photos/diagrams) to support text - ensure alt text is included.
  9. Use multimedia (audio/video) to support - include captions and transcripts.
  10. Run the application's accessibility checker.
